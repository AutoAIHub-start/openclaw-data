## 기술 스택
- **OpenClaw** — AI 에이전트 (현재 권한 문제로 제한적)
- **OpenCLI** — 설치 완료, 설정 진행 중
- **n8n** — 자동화 스케줄러 (설정 예정)
- **Discord** — 가상 오피스 (채널 설정 예정)
- **Claude API** — 에이전트 두뇌
- **Hugging Face** — Pixel 에이전트 이미지 작업
- **Replicate** — HF 과부하 대안
- **네이버 스토어팜 API** — 판매 데이터 수집

## HF 메모리 관리 전략
```
Github  → 코드, 설정파일, 프롬프트
HF Hub  → 모델 파일 (원래 여기 있어야 함)
Replicate → 큰 모델 실행 (HF Spaces 메모리 절약)
```
