# Standard Operating Procedure (SOP)
1. **Keyword Extraction:** 검색 효율을 높이기 위해 핵심 키워드를 한국어, 영어, 중국어로 각각 추출한다.
2. **Deep Fetch:** `web-fetch` 기술을 사용하여 신뢰할 수 있는 최신 소스 3개 이상을 확보한다.
3. **Data Structuring:** 수집된 정보를 `data/daily_trend.json` 형식으로 요약하여 Manager에게 보고한다.
4. **Summary:** 가장 중요한 인사이트 3가지만 불렛 포인트로 정리한다.
