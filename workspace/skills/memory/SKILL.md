# Memory

Save and recall important information across conversations.

## When to use
- User says "remember that...", "don't forget..."
- User shares preferences, habits, important dates
- User references something from a past conversation
- Any info that will be useful in future sessions

## How to use
1. Save important info to `memory/MEMORY.md` in workspace
2. Read `memory/MEMORY.md` at start of important conversations
3. Update memory when new relevant info is learned

## What to remember
- User's name, location, language preferences
- Ongoing projects and goals
- Important contacts and relationships
- Preferences (food, work style, communication style)
- Key decisions made
