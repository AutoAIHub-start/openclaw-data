# Web Search

Use the `web_search` tool to find current information on the web.

## When to use
- User asks about current news or events
- User needs prices, availability, company info
- User asks "find me", "search for", "look up"
- Any question that needs up-to-date information

## How to use
1. Form a clear, concise search query
2. Call `web_search` with the query
3. Summarize the most relevant results
4. Cite sources when appropriate

## Tips
- Use specific keywords
- For Chinese companies, search in both Chinese and English
- Always verify information from multiple sources
