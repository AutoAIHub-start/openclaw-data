# Research

Conduct deep, multi-step research using web search and web fetch.

## When to use
- User needs comprehensive analysis on a topic
- Finding suppliers, companies, prices, market info
- Comparing multiple options
- Any request needing more than one search

## How to use
1. Break the topic into 3-5 specific search queries
2. Search each query using `web_search`
3. Fetch full pages of most relevant results with `web_fetch`
4. Synthesize findings into a clear, structured report
5. Include sources and prices where relevant

## Output format
- Summary first
- Detailed findings with sources
- Comparison table when applicable
- Recommended next steps
