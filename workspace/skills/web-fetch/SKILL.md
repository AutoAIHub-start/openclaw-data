# Web Fetch

Read full content from specific URLs.

## When to use
- User shares a URL and asks to read/summarize it
- Research requires reading full article content
- Checking prices or availability on specific pages
- Extracting data from websites

## How to use
1. Use `web_fetch` tool with the target URL
2. Extract relevant information
3. Summarize key points clearly
4. Note the source URL in response
